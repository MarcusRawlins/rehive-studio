# Rehive Studio Marketing Site

A modern, dark-mode-first marketing site for Rehive Studio showcasing our software development services.

## Stack

- **Next.js 16** (App Router, Turbopack)
- **TypeScript**
- **Tailwind CSS** (v4)
- **shadcn/ui** components
- **Geist** fonts (Sans & Mono)

## Features

- ✨ Dark mode design with electric blue accents
- 📱 Mobile-first responsive layout
- 🎯 Smooth scroll navigation
- 💼 Portfolio showcase with ZipGolf flagship project
- 🚀 Modern tech stack display
- 📧 Contact form section
- ⚡ Fast performance (< 1s LCP target)

## Sections

1. **Hero** - "We build software that works." headline with CTAs
2. **Services** - Three service tiers (Websites, Web Apps, SaaS)
3. **Process** - Four-step workflow timeline
4. **Portfolio** - ZipGolf + 6 demo sites with live links
5. **Tech Stack** - Logo bar of technologies used
6. **Why Rehive** - Three key differentiators
7. **Contact/CTA** - Contact form + Calendly booking
8. **Footer** - Navigation and contact info

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

Visit [http://localhost:3000](http://localhost:3000) to view the site.

## Deployment

Configured for deployment on Render with custom subdomain: **studio.getrehive.com**

## Design Guidelines

- **Colors**: Dark backgrounds (#0a0a0a, #111111), white text, blue accent (#3b82f6)
- **Typography**: Geist Sans for body, Geist Mono for code/tech
- **Motion**: Subtle, professional animations
- **Tone**: Confident, technical, direct - no corporate jargon
